Sending GPS Coordinates from a Android device to a remote Mysql database

In this Project I have sent GPS coordinates(i.e. the Latitude and the longitude) of the user from a Android device to a Remote mysql database.This App records the latitude and longitude  from the GPS , shows it on the phone as a toast and then sends the data to the database.

The code for android app was done on Android studio and for the server part i used a free hosting service (freeunlimitedhosting.ml).

 

Before running this App initially 'enable the location mode' in your Phone and then for first time just 'open google maps' and 'then close it and open this app'.Now this app will get data from the GPS and will make Toasts for the user to realize that the app is working and then it will send the data to the backend server.

 

After creating a new project on Android Studio,In the manifest file add these two lines:

 

 

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>

 

<uses-permission android:name="android.permission.INTERNET"/>
